# INTOWORK — Templates de Refonte

Deux templates de page d'accueil inspirés de [Welcome to the Jungle](https://www.welcometothejungle.com/fr), adaptés à la plateforme INTOWORK et au marché africain.

## Ouvrir les templates

Double-cliquez simplement sur les fichiers `.html` dans votre navigateur.

---

## Template 1 — Magazine Éditorial
**Fichier** : `template1-magazine-editorial.html`

**Inspiration WTTJ** : Approche éditoriale, typographie volumineuse, contenu-first, ambiance humaine et moderne.

**Caractéristiques** :
- Hero split (texte + photo portrait africain) avec badge IA et double CTA
- Barre de recherche flottante (métier + localisation + secteur)
- Cards entreprises style "magazine" avec radius large et hover
- Grid offres d'emploi **avec photos** style magazine (aspect-video)
- Bande stats INTOWORK fond vert plein
- Section "Pourquoi INTOWORK" (3 features : Matching IA / CV Builder / Alertes)
- Témoignages sur fond sombre (3 cards)
- Footer complet avec newsletter inline
- **Ambiance** : Lumineuse, éditoriale, chaleureuse — fond blanc avec blobs verts/violets

---

## Template 2 — Search-First Minimaliste
**Fichier** : `template2-search-first.html`

**Inspiration WTTJ** : Hero sombre, barre de recherche centrale, ticker d'entreprises, approche fonctionnelle épurée.

**Caractéristiques** :
- Navbar noire (dark mode) sticky
- Hero **fond noir** avec gradient vert/émeraude sur le titre
- Barre de recherche centrale "glassmorphism" (3 champs + CTA) avec typewriter animé
- Ticker/scroll infini des logos entreprises (Orange, MTN, Ecobank...)
- Liste offres **horizontale** (cards plates avec badge IA Match violet)
- Section split 50/50 candidats (dark) / employeurs (vert clair)
- Section "Comment ça marche" avec grands chiffres en fond
- Tableau de tarifs 3 colonnes (Starter / Pro highlighted / Enterprise)
- Section newsletter fond vert
- Footer sombre avec liens pays
- **Ambiance** : Sombre, tech, professionnel, moderne — inspiration LinkedIn × WTTJ
